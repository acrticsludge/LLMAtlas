# Graphify Command

Follow the workflow in `me/tools/graphify-workflow.md`.

## Steps

1. Run graphify on the current repo
2. Read the output graph using query patterns from `me/tools/code-review-graph.md`
3. Surface: top dependency clusters, orphaned files, high-churn areas
4. Recommend which areas need attention before the next review
